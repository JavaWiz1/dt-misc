static determine_version(target_name: str, identify_src: bool = False) → str | Tuple[str, str][source]

Retrieve project version for distribution (or running codebase)

Version is determined by: - check importlib metadata - look for pyproject.toml - scanning calling stack root file newest python module

Parameters:

  • distrib_name (str) – Package distribution name. If distrib_name not found, version will be determined from pyproject.toml (if found) or from the newest .py file in the stack path starting at the calling program.

  • identify_src (bool, otional) – Return a string indicating how the version was determined. Defaults to False

Returns:

version or version, source. - version is in format major.minor.patch or YYYY.MM.DD. - source is one of ‘importlib.metadata’, ‘pyproject.toml’ or source filename.

Return type:

Union[str,Tuple[str,str]

Raises:

ValueError – if target_name is not str

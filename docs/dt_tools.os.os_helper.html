class dt_tools.os.os_helper.OSHelper[source]

Bases: object

Helper class for OS functions.

Supports windows and linux

Features:

  • OS/hardware detection (Windows/Linux/RaspberryPI)

  • Process detection (running in fg/bg)

  • Is executable in the path?

  • Is process running with admin/root permisssions?

Windowws: - is_admin - elevate to admin

Linux: - is_root

Raises:

OSError – On un-supported OS

Examples::

from dt_tools.os.os_helper import OSHelper

print(f’Is Windows: {OSHelper.is_windows()}) print(f’Is Linux : {OSHelper.is_linux()}) print(f’Is RPi : {OSHelper.is_raspberrypi()})
